# Тестирование на проникновение инфраструктуры FreeIPA (ОС Альт)

Оценка устойчивости к атакам на Kerberos по методике **OSSTMM**.
Репозиторий содержит пошаговые инструкции воспроизведения экспериментов на тестовом стенде.

> ⚠️ **Дисклеймер.** Все материалы предназначены **исключительно** для образовательных целей и проведения тестирования на проникновение в собственной изолированной лабораторной среде. Запуск описанных атак против систем, на которые у вас нет письменного разрешения владельца, незаконен. Авторы не несут ответственности за неправомерное использование.

## Методика оценки (OSSTMM)

Каждый эксперимент оценивается по трём метрикам Operational Security:

- **Visibility** — объём информации, доступной атакующему без преодоления защиты, и наблюдаемость атаки защитником.
- **Access** — количество и качество точек входа, барьер входа для атакующего.
- **Trust** — степень, в которой система выдаёт привилегии/информацию без явной аутентификации.
